Computermodel


CPU: De processor oftewel de Central Processing Unit is een van de belangrijkste componenten van een computer. Het voert namelijk alle berekeningen uit en wordt vaak het brein van een computer genoemd. Moderne processors zijn heel complex en worden door machines van miljoenen euro’s in elkaar gezet. Een slechte CPU kan leiden tot slome prestaties, bijvoorbeeld dat programma’s sloom worden opgestart. Een CPU is ook heel belangrijk voor multitasken want als je CPU slecht is kan dit ervoor zorgen dat als je veel applicaties open hebt dat alles vast loopt. Bij een CPU hoort een processor koeler die ervoor zorgt dat de processor niet oververhit, ik praat dan niet over een waaier over een waaier op je processor maar een volwaardige koeler die vastgeschroefd is tegen het moederbord. Normaal word dit gekoeld met een normale koeler, maar als je het budget ervoor hebt kun je ook een waterkoeler aanschaffen, deze zal de processor voldoende koelen. Als je de goede koeler hebt en de desbetreffende processor hebt kun je deze ook overclocken en een hogere ghz draaien. Een CPU heeft wat je noemt cores dit is meestal een indicator van hoe goed je CPU is hoe meer cores hoe beter je computer is en hoe beter je kunt multitasken of gamen voor CPU focussed games. Ik heb de processor van Ryzen genoemd de Ryzen 5 3600 wat voor zijn tijd een redelijk goede processor was.)


GPU: De videokaart, ook wel de Graphic Processing Unit, is ook een essentieel deel van je computer, dit handelt namelijk zoals de naam al zegt alle graphics. Een goede videokaart is meestal niet belangrijk voor de gemiddelde gebruiker, maar als je gamed is het van belang om een goede GPU te hebben, want dit zorgt ervoor dat je computer betere graphics kan draaien op een game. Sommige computers hebben geen GPU, maar een APU, dit is een processor die ook de berekeningen uitvoert die een GPU uitvoert, maar is over het algemeen slechter dan de gemiddelde GPU. Een APU zorgt er ook voor dat de CPU en GPU goed samen kunnen werken in sommige computers zitten hele goede processors maar slechte videokaarten wat zorgt voor een bottleneck en betekent dat je slechte GPU de CPU weerhoudt van zijn volledige potentie, omdat de GPU te slecht is. (Ik heb de videokaart van MSI genoemd 1050ti)


Moederbord: Het moederbord is het component dat alles met elkaar verbindt, de CPU met de GPU en de RAM en dergelijke. Zonder moederbord kan je computer niet werken. Een goed moederbord is belangrijk zodat al je componenten zich tot hun maximale potentie kunnen volbrengen. Bijvoorbeeld als je moederbord te oud is kan je RAM zich soms niet tot de maximale MHz kunnen volbrengen wat ervoor zorgt dat je pc in sommige instanties slomer is met multitasken bijvoorbeeld. Het is ook belangrijk dat je de goede combinatie van moederbord en CPU, want per merk is er een verschillende port en als je deze verbindt terwijl deze niet samen kunnen werken of dat je zelfs je hardware beschadigt en dit zijn geen goedkope grapjes.


HDD/SSD: Hardeschijven zijn belangrijk voor het langdurig bewaren van bestanden op je computer. Harddisks hebben ook in de meeste gevallen een grote capaciteit van 1 terabyte of zelfs meer. Een HDD is eigenlijk een oude manier van het bewaren van je bestanden en tegenwoordig gebruiken mensen vaker een SSD, omdat deze veel sneller zijn maar wel duurder. Hoewel deze sneller zijn nemen mensen deze meestal in lagere capiciteit om bijvoorbeeld hun besturingssysteem op te bewaren zodat hun computer sneller opstart of andere applicaties.


RAM geheugen: RAM, of Random Acces Memory is ook essentieel voor je computer omdat hier alle snelle berekingen op plaats vinden en je computer gebruikt je RAM geheugen om voor korte tijd dingen op te bewaren in tegenstelling tot een HDD die bedoeld is voor het bewaren van belangerijke bestanden voor lange tijd. RAM is in tegenstelling tot een HDD ook veel sneller en is ook belangerijk voor multitasken, omdat als je bijvoorbeeld een google chrome tab open hebt, dit RAM geheugen in.


Voeding: De voeding brengt alle elektriciteit naar je PC, meestal niet direct, want je plugt bijvoorbeeld een 24 pin in je moederbord en dit zorgt ervoor dat je RAM geheugen energie krijgt en soms brengt het zelfs energie naar je GPU als deze afhankelijk is van je moederbord. Een goede voeding is ook belangerijk zodat je computer niet explodeert en zodat je componenten genoeg energie krijgen om tot hun volledige potentie te werken sommige GPU’s gebruiken veel meer Watt dan andere videokaarten. Deze getallen kunnen lopen van 400 tot wel 1200 of zelfs in sommige gevallen nog hoger.
